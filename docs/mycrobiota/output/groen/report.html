Trimming, Alignment and Chimera Filtering

A number of trimming and filtering steps were performed in this analysis

  1. Trim with oligos file
  2. Trim with minlength=200, keepfirst=300
  3. Trim poorly called nucleotides
  4. Align unique sequences with an adaptation of the Bacterial SILVA SEED database release v119
  5. Remove chimeric sequences using the Uchime source code
